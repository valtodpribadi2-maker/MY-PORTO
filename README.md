# deskripsi website portfolio

🌟 Deskripsi Website Portfolio: utiiffat.dev

utiiffat.dev adalah sebuah website portfolio digital inovatif yang mengusung gaya desain Neo-Brutalisme modern dengan estetika playful khas platform kreator populer (seperti Saweria). Berbeda dengan portfolio statis konvensional, website ini dibangun berfokus sepenuhnya pada aspek interaksi pengguna, performa responsif, dan sentuhan micro-animation yang imersif.

🎨 Desain & Estetika Visual

Tema Neo-Brutalisme Modern: Penggunaan kombinasi warna pastel berenergi seperti soft off-whites, kuning cerah (#FFD166), biru muda, serta pink merona. Dilengkapi dengan garis tepi (3px border) berwarna hitam tebal dan bayangan padat (drop shadow) yang tajam, memberikan kesan berani, eksperimental, namun tetap bersih, rapi, dan terbaca (highly readable).

Trio Maskot Interaktif: Kehadiran maskot ikonik harimau, tupai, dan ayam jago yang memberikan identitas visual unik dan ramah.

Desktop & Mobile Friendly: Dioptimalkan secara penuh untuk kenyamanan akses di semua ukuran layar desktop maupun smartphone Anda dengan target sentuhan minimal 44px.

🚀 Fitur-Fitur Interaktif Unggulan

Dukungan & Buku Tamu Interaktif (Virtual Tipping Guestbook):
Pengunjung website dapat mensimulasikan pengiriman traktiran (seperti donat, kopi tubruk, boba, pizza, hingga roket booster) sembari meninggalkan pesan hangat.

Retro Sound Synthesizer: Setiap kali pesan terkirim, sistem akan memicu instrumen suara koin retro menggunakan Web Audio API langsung dari browser tanpa file audio eksternal.

Robot Text-to-Speech (TTS): Dukungan penuh otomatisasi suara berbahasa Indonesia dari Web Speech API yang akan membacakan nama pengirim dan isi pesan secara real-time.

Live Sandbox Editor (Pengubah Tulisan Langsung):
Dilengkapi dengan panel kontrol kustomisasi dinamis di bagian pojok kanan bawah. Pengunjung atau Anda sendiri dapat mengubah seluruh teks profil, biodata diri, tagline, lokasi, hingga rincian proyek secara instan langsung di layar dan menyimpannya secara lokal (Local Storage).

Showcase Proyek Berbasis Card Layout:
Kartu proyek interaktif yang memamerkan karya unggulan Anda lengkap dengan dialog detail (modal backdrop blur) untuk menjelaskan tech-stack dan dampak riil dari karya yang telah diciptakan.

Accordion FAQ Interaktif:
Fitur tanya-jawab kreatif yang sangat interaktif mengenai ketersediaan proyek freelance, tech-stack utama, serta cara memulai kolaborasi pengerjaan website bersama Anda.

⚙️ Senjata Utama (Tech Stack)

Frontend: React 19 (Hooks & Functional Components), Vite Bundler, TypeScript (Strong Typing).

Styling: Tailwind CSS (Utility classes) & Google Fonts (Inter, Space Grotesk, JetBrains Mono).

Animasi: motion/react (Transisi animasi halus, pop-up modal, & gestur tombol).

Interaction APIs: Web Speech API (Text-to-Speech Indonesia), Web Audio Context (Retro Synthesizer), & LocalStorage API.
